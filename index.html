tapnot

Title : tapnot
Project : Personal
Description : Android color matching game

Game play :
- The player will check if the inner box has the same color as the outer box, the player must tap the inner box fast, if they have the same color and the player will gain a point if not game over.
- The player will let the box slide as long as the inner box and outer box have different color and the player will gain a point, but if the player taps the inner box with different color as the outer box game over.
